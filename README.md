# EffFinder
EffFinder is an R package, for detecting the normalized efficiencies of single reference gene (RGs) and multi-RG combinations. This package includes two main functions Eff.singleFinder and Eff.indexFinder.
(1) Eff.singleFinder. The program of Eff.singleFinder is very time-consuming, because the large p-value data matrix needs to calculate through a series of computationally intensive procedures. To save time and computational resources, some intermediate variables were kept in global environments for further use.
(2) Eff.indexFinder. The runing time of Eff.singleFinder is very short. You can use Eff.indexFinder to detect the normalized efficiency of any normalization factor that you want to detect. The serial number of normalization factors can be found from a file named "Index file" generated by Eff.singleFinder function. So the Eff.singleFinder function should be run before Eff.indexFinder.
For more technical details about this algorithm, please see our publication: Jipan Zhang, Yongju Zhao: Normalized efficiency changes in reference genes (RG): a call to use multi-RG combination in qPCR experiments, 2021, Clinical Chemistry.
EffFinder version 1.0.0
